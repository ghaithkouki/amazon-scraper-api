# 🛍️ Amazon Product Scraper API (FastAPI + Playwright)

This API allows you to search Amazon US for product listings using a keyword and returns structured JSON data like ASIN, title, rating, image, and more.

---

## 🚀 Features

- Uses **Playwright** with headless Chromium to scrape search results
- Built on **FastAPI**
- Returns product info like `ASIN`, `title`, `price`, `rating`, and `image URL`
- CORS-enabled for frontend use
- Lightweight: returns up to 10 products
- Perfect for personal or educational projects

---

## 📦 Tech Stack

- **FastAPI** - Web framework
- **Playwright** - Headless browser automation
- **Uvicorn** - ASGI server
- **Railway / Replit / Render** - Hosting options
